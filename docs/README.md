在技术面试的时候，不管是Redis还是什么问题，如果你能举出实际的例子，或者是直接说自己开发过程的问题和收获会给面试官的印象分会加很多，回答逻辑性也要强一点，不要东一点西一点，容易把自己都绕晕的。
还有一点就是我问你为啥用Redis你不要一上来就直接回答问题了，你可以这样回答：
帅气的面试官您好，首先我们的项目DB遇到了瓶颈，特别是秒杀和热点数据这样的场景DB基本上就扛不住了，那就需要缓存中间件的加入了，目前市面上有的缓存中间件有 Redis 和 Memcached ，他们的优缺点……，综合这些然后再结合我们项目特点，最后我们在技术选型的时候选了谁。
如果你这样有条不紊，有理有据的回答了我的问题而且还说出这么多我问题外的知识点，我会觉得你不只是一个会写代码的人，你逻辑清晰，你对技术选型，对中间件对项目都有自己的理解和思考，说白了就是你的offer有戏了。

## 自我介绍
> 自我介绍遵循极简原则（1+3+1）第一句：我叫谁，结尾要重音，不能仅仅只介绍学校和专业。接着三句话：说明工作亮点与自己最近的工作经历。【突出满足岗位需求，以及自己的职业规划来表示自己愿意长期从事该岗位的工作】最后一句话为什么想要这个工作或者为什么胜任这个工作，因为没有这句话，面试官形不成对你的整体评价，同时自我介绍结束的地方就是面试正式开始的地方，面试官很容易将这个细节作为切入点，深入问下去，如果回答不上来一定惨不忍睹，所以一定要将话题吸引到最有自信的地方中。

1. 面试自我介绍：【非常重要，直接影响到后面的提问】
2. 自我介绍的时候主要是给面试官一个缓冲时间来熟悉你的简历，我们要通过自我介绍直击要害



一个案例：`我叫李建华，来自武汉大学计算机学院，华为公司是我们一直非常尊敬的公司，这一次我应聘的事网络研发的工作，从事网络相关的工作呢，是我一生的梦想，为此呢在研究生阶段，我很早就结合岗位的要求进行了准备，包括对各种协议的熟悉，常用算法的实践，在两年的时间里我也重点选择了网络相关的课题进行研究，希望能够加入公司从事网络相关的工作，谢谢`

拒绝是一件很难的事儿，有些人是天生不太会拒绝的。尤其当企业向你伸出橄榄枝的时候，怎么委婉而不失礼貌的回绝，也体现出你自己的水平和深度。婉拒的好，兴许能提升你在这个圈子里的美誉度。婉拒的不好，对自己的名声都会受影响。你可知道当你得罪一个HR的时候他的圈子基本就把你拉黑啦，一定不要质疑HR的影响力~！  本人觉得回绝offer 应该注意以下几点：        第一，应该体现出感谢，人家选择了你，是对你能力、人品等品格的一种认可，要有一个感恩的心态，感谢人家的信任和认可，毕竟HR从茫茫人海寻到你也是付出了辛苦的劳动的。        第二，告知原因，前提一定是真诚的，哪怕你的理由是因为钱少，离家远，或者感觉和企业气场不合等一些主观的原因，也要说的理由让人同情和理解。同理心很重要，让企业HR感受到确实于情于理不太合适来公司工作。        第三，表示遗憾，不要给人家感觉你是傲娇的拒绝，表示没有选择公司也是出于无奈，有理由的，但是态度要是肯定的，不能让人觉得优柔寡断，看不出你到底是真不想来还是假不想来。        第四，表示以后如果条件允许还是可以合作的，做事情不是能做绝，凡是给自己留条路，毕竟圈子就这么大，一切皆有可能。        第五，及时回复，不要等到报道的前一天或者两天才回复，用人单位都准备好让你入职了，你再回绝，就不太好了。         以下是搜集到的一些比较好的回绝offer的模板，供大家参考：                1.非常感谢贵公司对我的认可。但是由于（什么理由），不能接受贵公司的offer，我觉得非常遗憾。希望我们可以保持联系，以后如果有机会，希望能与贵公司由更紧密的联系。但这次我真的非常抱歉，还请贵公司理解。谢谢。        2.非常感谢贵公司愿意给我机会，并给我提供了适合我的职位。我非常珍惜这一次机会，但是由于（什么理由），没有办法成为公司的一员，我很遗憾。希望以后可以有合作的机会，还请贵公司理解。谢谢。        3.非常感谢贵公司的青睐，之前也对贵公司非常的向往，但我发现贵公司提供的平台与我的职业发展方向有一定的偏离，没办法接受贵公司的offer，十分抱歉。但希望以后能有幸与贵公司合作，还请贵公司理解。谢谢。        4.非常感谢贵公司发给我的offer邀请，这次机会对我来说十分珍贵，我也特别喜欢贵公司。但是由于工作地点，我思虑良久，还是说声抱歉，今日向您申请取消入职。给贵公司带来的不便还请谅解，希望贵公司能找到更加适合这个职位的人选，若今后有机会也非常希望能再次合作！真诚地说声抱歉！        5.接到贵公司发给我的offer邀请，我十分开心和荣幸。我思考很久，觉得自己在求职方面还有些事情考虑得不周全，思来想去还是决定向您申请放弃这次宝贵机会。特别抱歉给公司的工作带来不便，希望公司能有更好。        最后，奉劝大家，不是迫不得已不要回绝人家的OFFER，毕竟这是一种缘分，而且你心仪的另一家公司未必有这家好~ O(∩_∩)O哈哈~~ 所以大家回绝的时候一定要谨慎、谨慎、再谨慎~求职难，是难在了这并不是你每天都需要做的事，所以大家看起来缺经验，实际上是不了解每个细节的应对措施，再加上你在一个环境里工作了多年，外部是怎么变化的，你跟不上了！
